Analyzing Crime Patterns: Insights From Los Angeles Dataset

This project aims to analyze a crime dataset from Los Angeles since 2020, and develop an analytical solution using SAS Enterprise Miner. The objectives include identifying criminal hotspots, potential victim profiles, and analyzing common weapons used in crimes. Clustering is used in this project to analyse and cluster sectors such as crime categories, weapon usages and victim profiles. Moreover, each cluster is being analyzed with different supervised models to determine the model with the highest accuracy.

SAS Enterprise Miner Data Mining
